Deploy to Kubernetes') {\n    when DEPLOY_K8S = true\n    \n    Step 1: Terraform\n    ├─ terraform init\n    ├─ terraform validate\n    ├─ terraform plan -out=tfplan\n    ├─ terraform apply -auto-approve tfplan\n    └─ terraform output -json | jq .\n    \n    Step 2: Ansible\n    ├─ ansible --version\n    └─ ansible-playbook -i inventory playbook.yml -v\n    \n    Step 3: kubectl Deployment\n    ├─ kubectl set image deployment/medicine-stock\n    ├─ kubectl apply -f k8s/service.yaml\n    ├─ kubectl rollout status deployment/medicine-stock\n    └─ kubectl get pods -n medicine-stock\n}\n```\n\n### Trigger Configuration\n\n**Automatic**: When Jenkins job runs with `DEPLOY_K8S=true`\n\n**Parameters**:\n- `IMAGE_TAG`: Docker image tag (default: ${BUILD_NUMBER})\n- `PUSH_TO_REGISTRY`: Push to Docker Hub (default: false)\n- `DEPLOY_K8S`: Deploy to Kubernetes (default: false)\n\n---\n\n## Deployment Instructions\n\n### Option 1: Manual Deployment\n\n```bash\n# Step 1: Configure\ncp terraform/terraform.tfvars.example terraform/terraform.tfvars\n# Edit terraform.tfvars with your values\n\n# Step 2: Deploy Terraform\ncd terraform\nterraform init\nterraform validate\nterraform plan -out=tfplan\nterraform apply tfplan\ncd ..\n\n# Step 3: Deploy Ansible\ncd ansible\nansible-playbook -i inventory playbook.yml -v\ncd ..\n\n# Step 4: Verify\nkubectl get ns medicine-stock\nkubectl get all -n medicine-stock\n```\n\n### Option 2: Using Scripts\n\n```bash\n# Linux/macOS\n./scripts/deploy-terraform.sh\n./scripts/deploy-ansible.sh\n\n# Windows\n.\\scripts\\deploy-terraform.bat\n```\n\n### Option 3: Via Jenkins\n\n1.
